浮动的5个特点

块元素浮动之后,,如果不给宽度,宽度会由内容撑开

行内元素浮动之后,,可以设置宽高了,上下方向的margin也可以生效了

浮动元素 由于脱离文档流,所以无法撑起父元素的高度,,所以如果父元素没有指定搞得时候,高度就会（塌陷）丢失,对布局造成影响
